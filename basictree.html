EweEye TreeView

This is an example of a tree structure that was in the existing HTML at the time of the document ready event. The EweEye.TreeView scans the DOM looking for a UL tag with the class of "eweeye-treeview". If found, it proceeds to replace the existing structure with the EweEye.TreeView control where LI elements, with child UL/OL delements, become expandable "PlusMinus" nodes and others become "Primitive" object nodes.

Raw HTML

<ul class="eweeye-treeview">
        <li>Level 1, Item 1
            <ul>
                <li>Level 2, Item 1
                    <ul>
                        <li>Level 3, Item 1</li>
                        <li>Level 3, Item 2</li>
                        <li>Level 3, Item 3</li>
                    </ul>
                </li>
                <li>Level 2, Item 2</li>
                <li>Level 2, Item 3</li>
            </ul>
        </li>
        <li>Level 2, Item 2</li>
        <li>Level 2, Item 3</li>
    </ul>
